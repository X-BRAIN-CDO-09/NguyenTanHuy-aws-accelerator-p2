# Reflection W9

## Day A toi hoc gi

Toi hoc cach dong goi app Flask thanh Docker image, viet Kubernetes manifest va dung GitOps de quan ly trang thai cluster. Toi hieu rang Git nen la nguon su that, con ArgoCD la cong cu giup cluster bam theo Git.

## Day B toi hoc gi

Toi hoc observability khong chi la co dashboard. Can hieu logs, metrics, traces va cach chung tra loi cac cau hoi khac nhau. Toi cung hoc SLI, SLO, Error Budget va Burn Rate de alert gan voi trai nghiem nguoi dung hon.

## Day C toi hoc gi

Toi hoc progressive delivery va canary deployment giup giam rui ro khi release. Argo Rollouts co the dung Prometheus Analysis de quyet dinh tu dong tiep tuc hay dung rollout.

## Kho khan

Kho khan la phai ket noi nhieu cong cu voi nhau: GitHub Actions, Kubernetes, ArgoCD, OTel, Prometheus, Grafana va Argo Rollouts. Moi cong cu co YAML va khai niem rieng, nen can chia nho theo tung ngay de de hieu.

## Bai hoc rut ra

Bai hoc lon nhat la DevOps khong chi la deploy duoc app. Mot pipeline tot can co kiem tra, GitOps, kha nang quan sat, alert theo SLO va chien luoc release an toan. Khi cac phan nay ket noi voi nhau, he thong de van hanh va de demo hon.

## Toi se demo gi voi mentor

Toi se demo app Flask co `/health` va `/metrics`, workflow GitHub Actions build image, ArgoCD Application sync manifest, Grafana dashboard, Prometheus burn rate alert va Argo Rollouts canary co AnalysisTemplate dung success rate.
